# Projet intermédiaire cybersécurité

## Objectif du projet

Dans ce projet interne, j'ai décidé de séparer mon lab en deux parties :

1. Un environnement dédié à l'apprentissage de la cybersécurité
2. L'intégration de ces compétences dans mon lab principal contenant :
   - Windows serveur
   - GLPI
   - postes clients
   - infrastructure réseau

L'objectif de ce projet est donc d'apprendre les bases du pentesting en exploitant différentes vulnérabilités sur une machine volontairement vulnérable : Metasploitable 2.

La machine d'attaque utilisée est Kali Linux, qui contient les outils nécessaires pour effectuer :
- des scans réseau
- de la collecte d'informations
- l'exploitation de vulnérabilités

L'outil principal utilisé sera Metasploit Framework.
